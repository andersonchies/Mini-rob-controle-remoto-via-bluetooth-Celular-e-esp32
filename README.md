# 🚗 Mini Robô com ESP32 via Bluetooth

Projeto de um carrinho controlado por celular usando ESP32 e Bluetooth.

## ⚙️ Projeto
- Toda a programação foi realizada com o PlatformIO

## 📡 Tecnologias
- ESP32
- Bluetooth Serial
- Arduino / PlatformIO

## 🎮 Controle
Comandos recebidos via app Bluetooth:

| Comando | Ação |
|--------|------|
| F | Frente |
| B | Ré |
| L | Esquerda |
| R | Direita |
| G | Frente Esquerda |
| H | Frente Direita |
| I | Trás Esquerda |
| J | Trás Direita |
| Y | Buzina |

## 🔊 Extras
- Buzzer tocando música do Mario 🍄

## ⚙️ Como usar
1. Conectar no Bluetooth: `ESP32_BT`
2. Usar app de controle Bluetooth
3. Enviar comandos
